# Sea Surface Temperature

Sea Surface Temperature

## Usage

``` r
SeaSurfaceTemp
```

## Format

A data.frame with 15607 rows and 4 columns.

- lon:

  longitude

- lat:

  latitude

- sst:

  sea surface temperature in Kelvin

- type:

  defines training and test set
