- [`fitted(`*`<LMMsolve>`*`)`](https://biometris.github.io/LMMsolver/index.html/reference/fitted.LMMsolve.md)

  : Log-likelihood of an LMMsolve object
- [`mLogLik()`](https://biometris.github.io/LMMsolver/index.html/reference/mLogLik.md)
  : Function to obtain restricted log-likelihood and the first
  derivatives of the log-likelihood, given values for the penalty
  parameters
- [`deviance(`*`<LMMsolve>`*`)`](https://biometris.github.io/LMMsolver/index.html/reference/deviance.LMMsolve.md)
  : Deviance of an LMMsolve object
- [`predict(`*`<LMMsolve>`*`)`](https://biometris.github.io/LMMsolver/index.html/reference/predict.LMMsolve.md)
  : Predict function
- [`multinomial()`](https://biometris.github.io/LMMsolver/index.html/reference/multinomial.md)
  : Family Object for Multinomial Model
- [`diagnosticsMME()`](https://biometris.github.io/LMMsolver/index.html/reference/diagnosticsMME.md)
  : Give diagnostics for mixed model coefficient matrix C and the
  cholesky decomposition
- [`displayMME()`](https://biometris.github.io/LMMsolver/index.html/reference/displayMME.md)
  : Display the sparseness of the mixed model coefficient matrix
- [`effDim()`](https://biometris.github.io/LMMsolver/index.html/reference/effDim.md)
  : Function to get the Effective Dimensions.
- [`getHeritability()`](https://biometris.github.io/LMMsolver/index.html/reference/getHeritability.md)
  : Generalized heritability of a random term
- [`as.ginverse()`](https://biometris.github.io/LMMsolver/index.html/reference/as.ginverse.md)
  : Construct a ginverse Object from Precision Matrices
- [`makeGrid()`](https://biometris.github.io/LMMsolver/index.html/reference/makeGrid.md)
  : Create a data frame for use in making predictions.
